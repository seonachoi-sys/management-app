# CSS 규칙

## Prefix 규칙
- 이번 주 뷰: week-
- 토스트: toast-
- 사이드바: sidebar-
- 타임라인: timeline-
- 진척 현황: progress-
- 완료 하위업무: task-completed-

## 디자인 원칙
- 배경: 흰색
- 테두리: 0.5px solid #e5e5e5
- border-radius: 8px (카드), 10px (섹션)
- 폰트: 13-14px 본문, 11px 메타
- 색상:
  - danger: #FCEBEB / #A32D2D
  - warning: #FAEEDA / #854F0B
  - info: #E6F1FB / #185FA5
  - success: #EAF3DE / #3B6D11

## 금지
- 외부 차트 라이브러리 (CSS + SVG만)
- position: absolute 드롭다운 (fixed 사용)
