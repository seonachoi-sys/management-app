# 보고서 생성 가이드

## 보고서 3종 구조

### 주간 (팀 미팅)
기준: 이번 주 월~일
섹션: 완료 → 미완료 → 차주 예정 → 차주 이월 → KPI + 리드타임

### 2주 (대표이사)
기준: settings.ceoMeetingDates에서 선택한 날짜 ±14일
섹션: 2주간 완료 → 다음 2주 진행 → 결정 필요 → KPI + 리드타임
미팅 일정: 3/26 → 4/9 → 4/23 (격주 수요일)

### 월간 (팀 전체)
기준: 선택월 드롭다운 (기본: 지난달)
섹션: N월 완료 → N+1월 이월 → N+1월 예정
KPI 없음

## 주의
- 날짜 하드코딩 금지
- 완료 업무 기준: completedDate (completedAt 아님)
- 리드타임 null인 업무는 집계에서 제외
