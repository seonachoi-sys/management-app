# 버그 수정 가이드

## 수정 전 필수 확인
1. 해당 파일 전체 읽기
2. 영향받는 컴포넌트 파악
3. 수정 범위 최소화 (다른 기능 건드리지 않기)

## 자주 발생하는 버그 패턴

### 날짜 하루 밀림
원인: new Date('2026-03-19') → UTC 파싱
해결: new Date('2026-03-19T00:00:00') 로컬 파싱

### 담당자 필터 빈 결과
원인: 상위업무 assigneeName이 비어있어서 트리 깨짐
해결: 하위업무 담당자 기준으로 상위업무 포함 로직

### KPI 달성률 0% 리셋
원인: updateDoc 시 기존 targetValue 덮어씀
해결: getDoc으로 기존 값 읽어서 병합

### race condition
원인: tasks 클로저에 갇힌 상태
해결: tasksRef.current 사용

## 수정 후
- 타입 체크: npx tsc --noEmit
- 커밋 메시지: fix: [버그 내용] ([파일명])
- push 후 배포
